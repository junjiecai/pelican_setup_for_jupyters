Calculus(微积分)¶

In this section, we will cover how to perform baisc calculus tasks such as derivatives, integrals,litmis and series expansion.

Derivatives¶

To take derivatives, use diff()

basic derivatives¶

Pass in an expresison and one symbol with respect to which the derivative is applied.

In [49]:
def compare_derivative(expr,symbols, args = None, kwargs = None):
    after = diff(expr,symbols)
    
    print('Before derivative:',expr)

    print('After derivative :',diff(expr,x))
In [50]:

symbol = x

diff() can also be called as a method

In [51]:

higher order derivative¶

To take n order derivatives, pass the symbol n times or pass the symbol once and then follows it with n.

In [42]:
expr = x**4

compare_derivative(expr,x,x,x)

'Before evaluating:'
$$\frac{d^{3}}{d x^{3}} x^{4}$$ 
'After evaluating:'
$$24 x$$

You can achive the same calculation with the second syntax.

In [45]:
compare_derivative(expr,x,3)

'Before evaluating:'
$$\frac{d^{3}}{d x^{3}} x^{4}$$ 
'After evaluating:'
$$24 x$$

higher order partial derivatives.¶

Just pass the symbols in order, with the same syntax with single variable derivative.

In [46]:
expr = exp(x*y*z)

compare_derivative(expr, x, y, y, z, z, z, z)

'Before evaluating:'
$$\frac{\partial^{7}}{\partial x\partial y^{2}\partial z^{4}} e^{x y z}$$ 
'After evaluating:'
$$x^{3} y^{2} \left(x^{3} y^{3} z^{3} + 14 x^{2} y^{2} z^{2} + 52 x y z + 48\right) e^{x y z}$$

Or user number to control the derivatives order for each symbol

Integrals¶

indefinte integrals¶

Similiar with derivatives, integrals can be performed through integral() by function or method. To construct an unevaluated expression, initialize an Integral class and call doit() to evaludate the calculation.

To avoid duplication, let's define a function similiar with compare_derivatives()

In [55]:
def compare_integral(expr,*args,**kargs):
    args = args or []
    kargs = kargs or {}
    expr_diff = Integral(expr,*args,**kargs)
    display('Before evaluating:',expr_diff)    
    display('After evaluating:',expr_diff.doit())
In [57]:
expr = cos(x)

compare_integral(expr,x)

'Before evaluating:'
$$\int \cos{\left (x \right )}\, dx$$ 
'After evaluating:'
$$\sin{\left (x \right )}$$

definite integrals¶

To perform a definite integral, pass in a tuple of variable symbol, lower bound and upper bound.

In [59]:
expr = exp(-x)

compare_integral(expr,(x,0,oo))

'Before evaluating:'
$$\int_{0}^{\infty} e^{- x}\, dx$$ 
'After evaluating:'
$$1$$

Note, $\infty$ in Sympy is oo (double lower case 'O')

multiple integrals¶

To perform a multiple integral, pass multiple tuples of symbol variable and integral bounds.

In [61]:
expr = exp(-x**2 - y**2)

compare_integral(expr,(x, -oo, oo), (y, -oo, oo))

'Before evaluating:'
$$\int_{-\infty}^{\infty}\int_{-\infty}^{\infty} e^{- x^{2} - y^{2}}\, dx\, dy$$ 
'After evaluating:'
$$\pi$$

If Sympy cannot calculate an integral on an expression, it returns an unevaluated integral expression.

In [65]:
expr = x**x

compare_integral(expr,x)

'Before evaluating:'
$$\int x^{x}\, dx$$ 
'After evaluating:'
$$\int x^{x}\, dx$$

Limits¶

Similiar with derivatives, limits can be performed through limit() by function or method. To construct an unevaluated expression, initialize an Limit class and call doit() to evaludate the calculation.

To avoid duplication, let's define a function similiar with compare_limit()

In [70]:
def compare_limit(expr,*args,**kargs):
    args = args or []
    kargs = kargs or {}
    expr_diff = Limit(expr,*args,**kargs)
    display('Before evaluating:',expr_diff)    
    display('After evaluating:',expr_diff.doit())

To perform two-sided, pass the expression, variable symbol and the value to evaluate at. By default the limit is calculated from right to left with default setting dir = '+'.

In [75]:
expr = sin(x)/x

compare_limit(expr, x, 0)

'Before evaluating:'
$$\lim_{x \to 0^+}\left(\frac{1}{x} \sin{\left (x \right )}\right)$$ 
'After evaluating:'
$$1$$

To caluculate a limit from left to right. Pass dir='-'

In [76]:
expr = sin(x)/x

compare_limit(expr, x, 0,dir = '-')

'Before evaluating:'
$$\lim_{x \to 0^-}\left(\frac{1}{x} \sin{\left (x \right )}\right)$$ 
'After evaluating:'
$$1$$

Series Expansion¶

Sympy can calculate an asymptotic series expansion of a funciton around a point $x_0$ with respect to $O(x-x_0)^n$.

In [84]:
expr = sin(x)

expanded = expr.series(x,0,6)

display('Origin expression:',expr)

display('Expanded expression:',expanded)

'Origin expression:'
$$\sin{\left (x \right )}$$ 
'Expanded expression:'
$$x - \frac{x^{3}}{6} + \frac{x^{5}}{120} + \mathcal{O}\left(x^{6}\right)$$

To remove the order term, call removeO() method

In [85]:
display('Expanded expression without order term:',expanded.removeO())

'Expanded expression without order term:'
$$\frac{x^{5}}{120} - \frac{x^{3}}{6} + x$$
